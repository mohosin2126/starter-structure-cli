# __APP_NAME__

Next.js App Router starter with TypeScript, shadcn/ui-style primitives, and Tailwind CSS v4.

## Features

- Next.js App Router
- TypeScript
- Tailwind CSS v4 with CSS variables
- Local shadcn-style UI primitives
- `components.json` config and `cn()` utility
- Alias support with `@/*`

## Project structure

```text
.
|-- app/
|-- components/
|-- lib/
|-- public/
|-- .env.example
|-- components.json
|-- eslint.config.mjs
|-- next-env.d.ts
|-- next.config.mjs
|-- package.json
|-- postcss.config.mjs
`-- tsconfig.json
```

## Getting started

```bash
npm install
cp .env.example .env
npm run dev
```
